# Quantitative Financial Analysis: Derivable vs. Non-Derivable Data

Can be derived:

1. Basic price action:
   - Price range, price change, percentage price change for each time period and across the entire dataset
   - Daily/hourly open, high, low, close prices

2. Moving averages:
   - Simple Moving Average (SMA), Exponential Moving Average (EMA) for various periods

3. Volatility measures:
   - Standard deviation of price
   - Average True Range (ATR)
   - Bollinger Bands

4. Volume analysis:
   - Volume-weighted average price (VWAP) is already provided
   - On-balance volume (OBV)
   - Volume trends

5. Trade size analysis:
   - Average trade size for each period

6. Momentum indicators:
   - Relative Strength Index (RSI)
   - Moving Average Convergence Divergence (MACD)
   - Rate of Change (ROC)

7. Support and resistance levels

8. Basic trend analysis:
   - Identifying uptrends, downtrends, or sideways movements

9. Candlestick patterns

10. Intraday patterns and behaviors

11. Basic liquidity analysis using volume data

12. Simple returns calculations (hourly, daily)

Cannot be derived (would require additional data):

1. Market context (overall market index data, sector-specific indices)

2. Fundamental data (company financials, P/E ratio, book value, dividend yield)

3. Economic indicators (interest rates, GDP growth, inflation rates, unemployment figures)

4. Sentiment data (news sentiment, social media sentiment)

5. Order book data (bid-ask spreads, market depth)

6. Options data (option prices and volumes, implied volatility)

7. Short interest (short volume ratio, days to cover)

8. Analyst estimates (earnings forecasts, price targets)

9. Corporate actions (stock splits, dividends, mergers and acquisitions)

10. Related assets data (correlated stocks or assets, benchmark comparisons)

11. Macroeconomic events

12. Fundamental ratios (e.g., P/E ratio, EPS)

13. Market capitalization (would need shares outstanding)

14. Sector or industry classification

15. Dividend information

16. Ownership structure (institutional ownership, insider trading)
